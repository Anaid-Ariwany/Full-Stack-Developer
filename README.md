# Front-End Projects

This repository contains Full-Stack projects built following the [theodinproject](https://www.theodinproject.com/paths), [udemy-web-developer-bootcamp](https://www.udemy.com/course/the-web-developer-bootcamp/), 
[freecodecamp](https://www.freecodecamp.org/learn/) and [roadmap.sh](https://roadmap.sh/).

## Projects List 

[HTML Recipe Website](https://www.theodinproject.com/lessons/foundations-recipes#assignment), [HTML Single-Page CV Website](https://roadmap.sh/projects/basic-html-website)

[Basic HTML Website](https://roadmap.sh/projects/basic-html-website), [Landing Page](https://www.theodinproject.com/lessons/foundations-landing-page)
